# Nuxt Minimal Starter

Веб-приложение на Nuxt 4 для отображения каталога товаров и услуг компании. Проект включает три основных раздела: каталог товаров с фильтрацией по категориям, список агентов с рейтингами и отзывами, а также информацию о компании (расписание работы, местоположение, контакты). Приложение использует Vue 3, Tailwind CSS и современный стек технологий для создания адаптивного пользовательского интерфейса.
